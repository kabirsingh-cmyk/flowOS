## Creation Tools Reference

| Tool | What it creates | Used for |
|---|---|---|
| **Claude** | All copy, scripts, captions, subject lines, CTAs, threads | Every content type |
| **Runware** | Static images, ad creatives, thumbnails, pin visuals | Any format requiring a still image |
| **HeyGen** | UGC-style avatar video, talking-head, spokesperson | Reels, Shorts, TikTok, in-feed ads |
| **Runway** | Cinematic video, product video, B-roll, transitions | Premium video ads, YouTube, brand content |
| **Luma** | 3D-to-video, product spin, photorealistic scenes | Pinterest video, YouTube, product showcases |
| **ElevenLabs** | Voice-over, narration, podcast audio | Any video format requiring spoken audio |
